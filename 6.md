## 1. ORDER BY — Sorting Results

### Ascending (default)

```sql
SELECT * FROM students ORDER BY Name ASC;
-- ASC is the default, so this is the same as:
SELECT * FROM students ORDER BY Name;
```

### Descending

```sql
SELECT * FROM students ORDER BY Name DESC;
SELECT * FROM students ORDER BY Marks DESC;
```

### Multiple sort columns

```sql
SELECT * FROM students ORDER BY Marks DESC, Age DESC;
```

First sorts by Marks (highest first). When two rows have equal Marks, they are then sorted by Age (oldest first).

### Sort by column position

```sql
SELECT * FROM students ORDER BY 1 DESC;   -- sort by 1st column (ID)
SELECT * FROM students ORDER BY 2 DESC;   -- sort by 2nd column (Name)
```

The number refers to the position of the column in the `SELECT` list.

```sql
SELECT Name, Age FROM students ORDER BY 2 DESC;   -- sort by Age
```

> Using a position number larger than the number of selected columns is an error:
> `ORDER BY 20` → `Unknown column '20' in 'order clause'`

## 2. LIMIT — Restricting Row Count

```sql
SELECT * FROM students ORDER BY Marks DESC LIMIT 3;
```

Returns only the top 3 students by marks.

| ID | Name | Age | Address | Marks |
|---|---|---|---|---|
| 6 | Ali | 22 | Peshawar | 91 |
| 3 | Bushra | 21 | Islamabad | 91 |
| 1 | Asma | 20 | Rawalpindi | 85 |

---

## 3. UPDATE — Modifying Existing Rows

```sql
UPDATE students SET Name = "Ali" WHERE ID = 6;
```

Changes the name of the student with `ID = 6` from "Fahad" to "Ali".

> **Always include a `WHERE` clause** when using `UPDATE`. Without it, every row in the table is updated.

---

## Key Points

| Concept | Syntax | Notes |
|---|---|---|
| Sort ascending | `ORDER BY col ASC` | `ASC` is the default |
| Sort descending | `ORDER BY col DESC` | |
| Sort by multiple columns | `ORDER BY col1 DESC, col2 ASC` | Applied left to right |
| Sort by position | `ORDER BY 2` | Refers to column position in SELECT |
| Limit rows | `LIMIT n` | Returns at most *n* rows |
| Update rows | `UPDATE t SET col = val WHERE ...` | Always use WHERE |
| NULL in ORDER BY | | NULLs sort first in ASC, last in DESC (MariaDB) |
